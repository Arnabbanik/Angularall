Data Binding Start

Data Binding is the binding data from component to template & template to component vice-versa. This is two type 1. one-way Binding - either view template to component or component to view. Example: string interpolation & property binding for component class to view template. Event Binding for view template to component class 2. Twoway Binding - to both side
