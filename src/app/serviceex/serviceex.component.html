A sevice is a reusable typescript class that can be used in multiple components across your Angular application. service also used for communacating between other compoenents. if two components has a relationship of child and parent then they can passing data by input & output decorator but when components don't have parent-child relationship then we use services to communicate between them

Advantages of Services

  • Services are easier to debug and test
  • services helps to reuseability of code
